# 实验资源说明

本文档说明如何获取和使用本项目的实验资源，包括预训练模型、实验结果等。

## 📦 预训练模型

本项目使用的预训练模型存储在 `checkpoints/` 目录中。由于模型文件较大，GitHub 仓库可能不包含所有模型文件。

### ACAS Xu 模型

- **位置**: `checkpoints/acas/`
- **格式**: `.nnet` 文件
- **说明**: ACAS Xu ReLU 网络模型，用于 ACAS 实验

### ACAS Tanh 模型

- **位置**: `checkpoints/acas_tanh/`
- **格式**: `.pt` 文件 (PyTorch 模型)
- **说明**: ACAS Xu Tanh 网络模型，用于 Tanh 网络实验

### RocketNet 模型

- **位置**: `checkpoints/RocketNetReLU/`
- **格式**: `.pt` 文件 (PyTorch 模型)
- **说明**: RocketNet ReLU 模型，用于 RocketNet 实验

### 获取模型

如果模型文件未包含在仓库中，您可以通过以下方式获取：

1. **从论文作者处获取**: 请联系论文作者获取模型文件下载链接
2. **自行训练**: 参考 `src/construct_acas_tanh.py` 了解如何构建 ACAS Tanh 模型
3. **使用公开数据集**: ACAS Xu 模型可以从 [ACAS Xu 官方资源](https://github.com/verivital/nnv) 获取

## 📊 实验结果

论文中的实验结果可以通过运行实验脚本复现。所有结果将保存在 `results/` 目录中。

### 结果文件格式

- **CSV 文件**: 包含实验指标（Ls, Us, Us-Ls, time 等）
- **JSON 文件**: 包含详细的实验配置和结果
- **可视化图像**: PNG 格式的图表和可视化结果

### 复现实验

要复现论文中的实验结果，请：

1. 确保已安装所有依赖（见主 README）
2. 确保模型文件在 `checkpoints/` 目录中
3. 运行相应的实验脚本（见主 README 的"快速开始"部分）
4. 结果将自动保存到 `results/` 目录

## 🔧 实验配置

实验的关键参数在脚本中已固定，以确保可重现性：

- **随机种子**: 1024
- **采样参数**: 见各脚本中的配置
- **树深度**: 见各脚本中的配置
- **概率阈值**: 见各脚本中的配置

## 📝 注意事项

1. **模型文件大小**: 某些模型文件可能很大（数百 MB），请确保有足够的存储空间
2. **GPU 内存**: 某些实验可能需要较大的 GPU 内存（16GB+）
3. **运行时间**: 完整实验可能需要数小时，请耐心等待

## 🐛 问题反馈

如果您在获取或使用实验资源时遇到问题，请：

1. 检查主 README 中的"常见问题"部分
2. 联系论文作者
3. 在 GitHub Issues 中报告问题
